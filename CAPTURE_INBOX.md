# CAPTURE INBOX — unfiled intent for the weekly planner

Drop brain-dumps here (or have the `capture` skill write here). The Monday
weekly-planning task reads this file: any entry newer than the last plan is
folded into that week's objectives/KPIs, then you decide what to file to the
spine. This is a staging surface for what's in your head but NOT yet in the
record — REM captures what happened; this captures what you intend.

Format per entry (capture skill emits this shape):
## <ISO date> — <short title>
**Projects & Ideas:** …
**Tasks:** …
**Connections (verified):** …
_status: unfiled | folded-into-plan | filed-to-spine_
